2017部落格改版

Hans Liu on 2017-06-30 category 寫程式 tags Blog comments Comments fb Comments

在參加2017的 PyCon 時，一邊聽台上的講者在分享，一邊在滑著自己的部落格，突然發現自己部落格的畫面有點陽春，就萌生替部落格改版的想法，所以在 PyCon 結束後就重新檢視了自己的部落格，研究一些現有的部落格設計，希望能在保留主要元素的前提下，讓部落格在文章和照片的閱讀上可以更加舒適，並且讓網站的結構可以更加簡潔。

這次改版前前後後花的時間有兩週多，比原本預想的時間還要多了一點，因為久久才改一次，一堆設計都忘光光，不過從成果來看自己還蠻滿意的，希望能帶來更好的瀏覽體驗。

主要的更新如下：

  1. 更完善的 css3 設計，同時移除對 bootstrap 的依賴
  2. 修正 categories 和 tags 網頁打不開的問題
  3. 調整 header 和 sidebar 的結構，強化 RWD 的支援
  4. 加入新版的 sharethis 功能
  5. python 升級到 3.5，同時 pelican 升級到 3.7.1
  6. 加入 pipeline pelican publish 的功能，這樣更新文章可以做到自動部屬
