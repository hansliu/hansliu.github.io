新加坡永久居民 PR 申請與通過心得

Hans Liu posted on May 03, 2021 |

在我申請新加坡 PR 的時候，申請流程已經全面e化，在申請時是不用繳交任何的實體文件，全部都能透過網路完成。

只要在事前準備好相關資料，然後到 ICA 網站依序上傳各個文件的電子檔，最後繳交100新幣的手續費即可。

不同身分申請 PR 要準備的文件可能會有所差異，詳細可以參考 ICA 網站。

因為我是 EP，我當時上傳的資料如下：

  1. EP 正反面電子檔
  2. 護照資訊與入境簽證電子檔
  3. 大學/最高學歷中英畢業證書電子檔
  4. 英文出生證明電子檔（可以到出生醫院或是區公所申請）
  5. Annex A 電子檔（請公司提供）
  6. Appendix - 1 電子檔
  7. Declaration of Applicant(s) (PTS) 電子檔
  8. 最近6個月的 Payslip 電子檔

之後就是漫長的等待，有關審核時間和通過規則這點，在網路與生活上可以聽到各式各樣的都市傳說。例如有人說要多做義工和志工，還有的說法是看你繳稅的次數和對國家的貢獻程度，甚至有一說是看你能否增產報國洗基因？等等的，不過這些說法都沒有事實根據，所以也僅能當做參考。

總之我從申請到通過等了六個多月。

經過等待之後，如果 ICA 的結果是通過，接著會收到一個信封，這個信封有點厚度，用平信投遞。裡面的附件都能在 ICA 網站下載到，但最重要的 IPA Letter 是無法下載的，所以如果等了很多天沒有收到，務必要打電話給 ICA 要他們補寄。

信的內容就是知會你已經通過了，請接著在一個月內準備好資料並完成必要的手續。

其中要我準備的資料如下：

  1. 有效 EP
  2. 有效護照
  3. 六個月內護照用大頭照兩張（新加坡護照用的規格）
  4. 完成開通 MediShield Life 和 CareShield Life Health
  5. 完成 Statutory 的檢驗報告（其實就是抽血驗 HIV 和照胸部 X 光驗肺結核）
  6. 完成申請 PR IC Card 的表格
  7. 完成工作證明 EP152 的表格（請公司提供）
  8. 完成申請 REP 的表格
  9. 所有申請時提供的電子檔正本（Payslip 是要最近6個月的）

以上資料都準備完畢後，就可以到 ICA 網站預約時間，其中工作證明 EP152 的表格由公司提供後只有兩週的有效期限，所以在預約時間的時候要注意時效，再來就是當天到現場完成所有的文件審核。

PR application 在 ICA Building 的五樓，到現場抽號碼牌等叫號，負責人員會依序把你的申請文件正本都確認過一遍，如果資料都沒有問題，你的 EP 當場就會被收走並注銷，並在現場重新拍照跟壓指紋，然後會發給你臨時的 PR collection slip 用於領取 PR 的卡片，同時會發給你有效的 Entry Permit 和 Re-Entry Permit。

完成手續的隔天即可到 ICA 網站預約領卡的時間，到時就是拿著 PR collection slip 去領取正式的 PR 卡片。

因為 PR 是新的 NRIC，所以過去用 FIN 申請的服務會需要做資料更新，我更新的資訊如下：

  1. Employee（公司要求提供 Entry Permit）
  2. SingPass（寄信通知更新 NRIC，重新設定 2FA 認證）
  3. TraceTogether（手機重新安裝）

成為 PR 後對於外勞來說最主要的差別還是在工作的自主權，有 PR 代表你將不用受限於公司發放的工作簽證，在找工作上面更加的自由，與公司談判也能更大聲。另外的差異則會是在收入上，薪資所得的一部分將被強迫扣繳 CPF，同時公司也要強迫提撥 CPF。值得提醒的是，當你成為 PR 的第一年，因為 CPF 帳戶金額還不足以扣繳政府強制購買的 Medishield Life 醫療保險，所以會收到要求補繳 CPF 的催繳通知信，其餘的日常生活則不會有太大的差異。
