2014大阪京都奈良五天四夜《下》

Hans Liu posted on Nov 16, 2014 |

前情提要 2014 大阪京都奈良五天四夜《中》

Day 4

奈良斑鳩一日券 -> 奈良 -> 春日大社 -> 水谷神社 -> 三月堂/二月堂 -> 東大寺 -> 奈良公園 -> 奈良東向商店街 -> 難波/道頓堀/心齋橋 -> 飯店

經過第三天的溫泉充電後，第四天行程的主要目標就是奈良與鹿，使用的票券是第一天就事先買好的奈良斑鳩一日券大阪市交通局版，當日可以不限次數的搭乘大阪市地鐵與奈良當地的公車。

到達近鐵奈良後，我們是先搭公車直達春日大社本殿，然後才從春日大社往近鐵奈良站玩回來。

奈良隨處可見的鹿鹿警告標語，因為鹿群們只要看到你手拿鹿餅或是食物就會一窩蜂的湧上來，而且真的會有突入衝擊和口水攻勢等危險舉動，所以如果有帶小朋友去的話，請務必注意小朋友的安全。

餵食鹿的鹿餅，根據同行友人的品嚐？證實鹿餅沒有想像中的脆和好吃...

請給我食物。

春日大社 Wikipedia

春日大社（かすがたいしゃ）是一座位於日本奈良縣奈良市奈良公園內的神社。舊稱春日神社。建於和銅二年（710年）。社內供奉的神明包括武甕槌命、經津主命、天兒屋根命和比賣神。神社例祭日為3月13日（春日祭）。建設者為藤原不比等。作為古都奈良的文化財的一部分而被登入世界文化遺產之列。為二十二社之一。

在春日大社因為正值11月初，我們遇到許多日本家庭帶著小朋友來進行七五三節的祈福參拜，身著日本傳統正式服裝的小男生非常的帥氣，小女生則是非常的美麗。

來到日本當然也要入境隨俗寫塊繪馬祈福大家都能身體健康與平平安安。

水谷神社。

銀杏未黃，但已經相當漂亮了。

東大寺 Wikipedia

東大寺，位於日本奈良縣奈良市雜司町，是華嚴宗大本山，南都七大寺之一，距今約有一千二百餘年的歷史。1998年作為「古都奈良的文化財」的一部分被列為世界文化遺產。

佛寺是728年由信奉佛教的聖武天皇建立的。東大寺是全國68所國分寺的總寺院。因為建在首都平城京以東，所以被稱作東大寺。另外有西大寺。

東大寺大佛殿，正面寬度57米，深50米，為世界最大的木造建築。大佛殿內，放置著高15米以上的大佛像——盧舍那佛。東大寺院內還有南大門、二月堂、三月堂、正倉院等。南大門有很著名的雙體金剛力士像。二月堂能夠俯視大佛殿和眺望奈良市區。

延著水谷神社往三月堂/二月堂的方向走，逛完三月堂/二月堂後繼續往奈良公園走，就可以走到東大寺，寺內除了大佛真的很大之外，最有名的就是位於大佛的右後側，有一根厚重的樑柱，底下有一個洞孔，俗稱大佛的鼻孔。聽說要是能夠順利穿過大佛的鼻孔，即可受到大佛的平安保佑。

有穿過有保佑。

離開東大寺，回到奈良公園中與楓葉和銀杏為伍，四周可見鹿群環繞著你，不知不覺就讓人忘卻時間流逝，帶串東大寺門口好吃的醬油糰子，找地方坐下來慢慢品嚐，是個可以讓人發呆一整天的好地方。

結束東大寺與奈良公園，回到近鐵奈良站買柿葉壽司和中谷本舖的麻糬當午餐，逛逛奈良東向商店街。之後回到難波/道頓堀/心齋橋開始真正的大採購，把衣服藥妝雜貨眼鏡一口氣買齊全，迎向回程的第五天！

Day 5

大阪市地鐵 -> 黑門市場 -> 難波/道頓堀/心齋橋 -> 飯店 -> 南海電鐵 -> 關西機場

最後一天的行程相當簡單，早上從飯店 Check Out 之後，行李先寄放在飯店，然後就是去黑門市場享受美食，交通一樣是使用 ICOCA 。

一早先去趟日本郵局寄明信片。

位於日本橋站附近的黑門市場，是有規劃的觀光市場，有很多提供立吞的攤販（現買現煮現場站著吃），平價的食材到高級的食材應有盡有，端看你口袋的預算有多少，我是覺得我們吃喝的還算少，有香濃豆乳、壽司、烤扇貝、烤小章魚、國產和牛、草莓奶昔等...

吃飽喝足後，回飯店拿行李，搭乘南海電鐵空港急行回關西機場，帶著豐富的紀念品滿載而歸，結束好玩的大阪京都奈良五天四夜。

回家後整理出來滿滿的發票與票券。

這次將行程依照鬆緊程度交錯排列看來收到很好的效果，雖然經過連續多天的步行，到了最後一天身體還是非常疲勞，但每一天的精神都還算不錯，也沒有發生無法在走下去的鐵腿危機。下次應該會繼續依照類似的方法來排出國自由行的行程。
