2014大阪京都奈良五天四夜《上》

Hans Liu on 2014-11-15 category 攝影記錄 tags 日本 大阪 京都 奈良 comments Comments fb Comments

雖然不是第一次出國自由行，但這次是第一次自己完成規劃的自由行。因為這次比較晚決定旅行的時間與細節，所以我是直接買旅行社的團體自由行，機票則是選日本航空 JAL 中午出發，晚上回來的班機，景點規劃上，五天的行程主要包含一天的京都，一天的奈良和一天半的大阪。

五天的行程大綱與重點吃喝

  • Day 1
    關西機場 -> 南海電鐵 -> 飯店 -> 難波/道頓堀/心齋橋 -> 飯店
    吃飛機餐、拉麵、章魚燒
  • Day 2
    京阪本線 -> 東福寺 -> 伏見稲荷大社 -> 清水寺 -> 二年坂/三年坂 -> 寧寧之道/石屏小路 -> 八坂神社 -> 花見小路通 -> 京都祇園四條通 -> 飯店
    吃炒麵、章魚燒、鰻魚飯、蕎麥麵、湯豆腐、烤肉串、牛丼
  • Day 3
    大阪周遊券 -> 通天閣 -> 新世界 -> 日本橋 -> 梅田藍天大廈 -> 梅田 -> 飯店 ->浪速之湯 -> 飯店
    吃飯店早餐、炸串、炸豬排
  • Day 4
    奈良斑鳩一日券大阪市交通局版 -> 奈良 -> 春日大社 -> 水谷神社 -> 三月堂/二月堂 -> 東大寺 -> 奈良公園 -> 奈良東向商店街 -> 難波/道頓堀/心齋橋 -> 飯店
    吃飯店早餐、醬油糰子、壽司、麻糬、拉麵
  • Day 5
    大阪市地鐵 -> 黑門市場 -> 難波/道頓堀/心齋橋 -> 飯店 -> 南海電鐵 -> 關西機場
    吃飯店早餐、生魚片、壽司、國產和牛、飛機餐

Day 1

關西機場 -> 南海電鐵 -> 飯店 -> 難波/道頓堀/心齋橋 -> 飯店

JAL 在台灣出境的時候是在桃園機場第二航廈，我對第二航廈的印象是有點小...

去程搭乘的飛機 JAL 816

飛機上有提供 drinks menu，可以喝到招牌的奇異果汁，綠綠的一杯，酸酸甜甜的口感還不錯。

JAL 的餐點就我對飛機餐的評價來說是好吃的！

到達關西機場，完成入境手續後，我就先在機場將未來幾天會用到，而且可以事先購買的票券一次買齊，包含大阪出張套票、大阪周遊一日券、ICOCA，在關西機場唯一買不到的是奈良斑鳩一日券大阪市交通局版，不過這票券也只要到大阪之後就可以在大阪的地鐵站買到了。

南海電鐵乘車券，進入閘口用。

出張票的一日乘車券，我是拿來在南海電鐵轉乘大阪地下鐵的時候用。

ICOCA 是當悠遊卡用，當需要買地鐵票的時候，可以節省買票的時間。

大阪出張套票、大阪周遊一日券、ICOCA。

我們四天晚上住宿的飯店是『大阪淀屋橋三井花園飯店』，個人覺得是一間很棒的飯店，在出發前幾天臨時收到旅行社這邊的通知，變成旅行中間需要更換房間（代表有一天上午要先 Check Out ，下午再重新 Check In），不過換來的優惠就是有提供早餐（我們原先沒有訂早餐），雖然在整理行李上麻煩了點，但後來覺得也不錯，因為飯店提供的早餐很好吃。

我們到達飯店 Check In 的時間已經是晚上，所以放完行李就坐地鐵到難波/道頓堀/心齋橋，開始第一晚的購物與吃喝。日本真的是個很好逛很好買的地方，在消費品的售價上面真的非常的具有吸引力，而且會搭配各式各樣的行銷活動，再加上護照退稅的優惠，只會讓人感覺到花錢很值得！

第六代江崎固力果跑跑人，原本有點期待看到「工事期間限定企劃」的綾瀨遙跑跑人，可惜就在我們出發前兩週，綾瀨遙跑跑人下台一鞠躬了...

日本章魚燒真的是相對便宜、大顆又美味的小吃。

不知曾幾何時一蘭拉麵變成深受觀光客喜愛的一家拉麵店了，無時無刻都在排隊，因為一蘭拉麵是24小時營業，建議要吃的話可以等商店街都逛完在來就好。

第一天的行程就在吃飽喝足後到尾聲，坐地鐵回飯店休息準備明天前往京都賞楓去。
