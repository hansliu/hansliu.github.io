2014大阪京都奈良五天四夜《中》

Hans Liu posted on Nov 16, 2014 |

前情提要 2014 大阪京都奈良五天四夜《上》

Day 2

京阪本線 -> 東福寺 -> 伏見稲荷大社 -> 清水寺 -> 二年坂/三年坂 -> 寧寧之道/石屏小路 -> 八坂神社 -> 花見小路通 -> 京都祇園四條通 -> 飯店

第二天一大早起床，使用 ICOCA 搭京阪本線，直奔東福寺賞楓葉。

東福寺 Wikipedia

東福寺（とうふくじ）是位在日本京都府京都市東山區本町的寺院，臨濟宗東福寺派大本山。山號「慧日山」（えにちさん）。本尊釋迦如來、開基（創立者）為九條道家、開山（初代住持）為聖一國師圓爾，在京都五山中位居第四位。境内的通天橋附近是紅葉名所。

因為是11月初，東福寺的楓葉還在變色中，但是我們仍然在通天橋就感受到楓葉的美！有機會一定要來看看楓葉完全見傾的樣子。

逛完東福寺之後，我們是步行前往伏見稲荷大社，以男生腳力來估計的話，約15分鐘就可以走到伏見稲荷大社。

伏見稲荷大社 Wikipedia

伏見稻荷大社位於稻荷山的山麓，在傳統上整個稻荷山的範圍都被視為是神域（聖地）的範圍。伏見稻荷大社主要是祀奉以宇迦之御魂大神為首的諸位稻荷神，自古以來就是農業與商業的神明，除此之外也配祀包括佐田彥大神、大宮能賣大神、田中大神與四大神等其他的神明。由於每年都有大量的香客前來神社祭拜求取農作豐收、生意興隆、交通安全，使得該神社成為京都地區香火最盛的神社之一。另外，起源於江戶時代的習俗，前來此地許願的人們往往會捐款在神社境內豎立一座鳥居來表達對神明的敬意，使得伏見稻荷大社的範圍內豎有數量驚人的大小鳥居，而以「千本鳥居」之名聞名日本全國乃至於海外。捐款豎立鳥居的單位包含個人、公司行號乃至於各地的商會組織，目前現存的鳥居，最早可以追溯到明治年間。

在神社的分類上，伏見稻荷大社屬於式內社（被延喜式神名帳所收錄的神社）中的名神大社（祭祀知名神明、自古以來就非常靈驗的神社）等級，為二十二社之一，舊時神社社格屬官幣大社等級。

聞名日本與海外的千本鳥居，曾多次被電影和動畫當做取景的參考依據。

千本鳥居和狐狸的繪馬。

在伏見稲荷大社走走千本鳥居，買買狐狸吊飾和御守，並且品嚐好吃的炒麵、章魚燒和鰻魚飯後，繼續搭乘京阪本線到清水五條，開始下午的清水寺之旅。

清水寺 Wikipedia

清水寺是一座位於日本京都府京都市東山區清水的寺院，於778年（寶龜9年）前後由延鎮上人起造。清水寺的山號為音羽山，主要供奉千手觀音，原本屬於法相宗這一宗派但目前已獨立，成為北法相宗的大本山。

清水寺與北山鹿苑寺（金閣寺）、嵐山天龍寺等同為京都境內最著名的名勝古蹟，一年四季前來朝拜的香客或來訪的觀光客是絡驛不絕。除此之外，清水寺也與石山寺（滋賀縣大津市），長谷寺（奈良縣櫻井市）等寺廟並列，自古以來就一直以嗣奉觀音的廟宇而為人所熟知，也是西國三十三箇所（在關西地區參拜33所觀音廟的巡訪活動）中第16所寺廟。自平安時代以來，就經常出現於日本文學作品當中。1994年，清水寺以身為古都京都的文化財之一部分，列名至世界文化遺產中。

清水寺除了看那堪稱世界奇觀也不為過的神奇木造建築之外，最主要還是逛二年坂、三年坂、寧寧之道、石屏小路等充滿古樸風味的街道巷弄。在人群中處處可見穿著和服的觀光客與日本人，角落會看到在努力宣傳的人力車伕，搭配兩旁林立的紀念品小店，以及各式各樣的傳統木造房舍，那氛圍真的是非常的舒服，令人流連忘返。

順著走過八坂神社就可以走到京都祇園四條通。

京都祇園四條通周遭是京都夜生活的重點區域，其中最有名的當然是會有藝妓出沒的花見小路通，還有百貨公司林立的京都車站，以及滿是居酒屋的木屋町通。我們很幸運的有在花見小路通遇見正在趕場的藝妓，不過保持禮貌沒有做攝影的行為。逛完祇園四條通後，一樣搭乘京阪本線返回飯店，結束一整天的鐵腿之旅。

Day 3

大阪周遊券 -> 通天閣 -> 新世界 -> 日本橋 -> 梅田藍天大廈 -> 梅田 -> 飯店 ->浪速之湯 -> 飯店

由於第二天的行程很累，所以我在第三天的安排上就相對輕鬆許多，上午去通天閣，中午在新世界吃炸串，下午逛逛日本橋，晚上去梅田看夜景加上吃晚餐，然後直接在梅田逛街購物，最後再去泡浪速之湯溫泉消除幾天來的疲勞，第三天的的交通與門票都靠大阪周遊一日券。

通天閣 Wikipedia

通天閣是位於日本大阪府大阪市浪速區新世界商業街的一個瞭望塔。為日本的登錄有形文化財產。具體地點是大阪府大阪市浪速區恵美須東1-18-6。

通天閣這個名字的意思是「直通天空的高建築物」，命名者是明治初期的儒學者藤澤南嶽。通天閣是由建築師內藤多仲所設計，他也是東京鐵塔的設計者。現在的通天閣完成於1956年（昭和31年），是第二代通天閣。塔高100米，含避雷針的話則有103米。塔上有電機大廠日立的廣告。

從歷史的角度來看，通天閣已經失去許多的色彩，不過日本人還是把通天閣裝飾的充滿娛樂性質，除了有許多通天閣限定的商品，還有展出一些有趣的大阪歷史文物。

非常可愛的大阪燒 Hello Kitty。

從通天閣往下看。

歷代固力果跑跑人看板。

當天剛好有固力果搭配1111的宣傳活動，拍下可愛的固力果 Show Girls。

離開通天閣，吃完午餐後轉戰日本橋，看看路上宣傳的女僕與逛逛日本動漫相關商品店，玩玩夾娃娃和轉蛋，傍晚時在移動到梅田，本來期待去梅田藍天大廈可以好好一睹大阪的夜景，可惜第三天的天氣狀況一直都很差，一整天都在細雨濛濛，導致我們在梅田藍天大廈的戶外夜景區無法欣賞到最完美的夜景。

梅田藍天大廈 Wikipedia

梅田藍天大廈（日語：梅田スカイビル）是日本大阪府大阪市北區新梅田城內的摩天大樓。地上40層樓、地下2層，高度約為173公尺，1993年3月完工。由建築師原廣司設計。由於獨特的建築外型，已成為大阪地區的地標和觀光勝地之一。

梅田藍天大廈電扶梯。

不過運氣還不錯有看到梅田藍天大廈的聖誕樹在試點燈。

在梅田瘋狂採購之後，我們決定先回飯店放下手上的大包小包，換上輕裝帶上毛巾，再前往浪速之湯泡溫泉，紓緩這幾天四處奔波的雙腿，雖然最後有點刺激的要趕末班車回飯店，但依然替第三天做了美好的收尾。
