使用 name.com 申請新域名

by 996 / Hans
in 胡言亂語 on 2014-03-26 | tags: domain name github page

搬家的第二步驟，就是決心要註冊一個新域名，想著趁這次機會好好的養一個專屬域名，事前也上網搜尋了許多部落客的經驗，大致上整理出國內國外的數家網域名稱代理商。

  1. pchome - http://myname.pchome.com.tw
  2. GoDaddy - http://www.godaddy.com
  3. name.com - https://www.name.com
  4. eNom - http://www.enom.com
  5. namecheap - https://www.namecheap.com
  6. yahoo.com - https://smallbusiness.yahoo.com

因為我沒有要使用 .tw 的頂級網域名稱，所以就放棄國內域名商 pchome，打算挑選不論是網站流量或是廣告折扣都打很兇的 GoDaddy，而且網路上的中文教學資訊也相當的豐富。

可惜的是，實際使用 GoDaddy 的情形讓我大失所望，先是註冊 GoDaddy 的過程中就讓我感受到了重重妨礙，它的使用者介面相當不具有親和力，而其註冊時的安全考量也相當的令我困惑（例如註冊密碼限制要9位數以上），以及龐大的彈出式廣告與配套方案，都讓我感到不適應，而且其中最令人稱讚的 coupon 優惠，在加上 privacy service 之後的價格也變得沒有什麼競爭力...

終於在我好不容易註冊完成，順利登入 GoDaddy 的首頁後，沒想到會員後台也會有彈出式視窗，這一點也讓我正式宣告放棄使用 GoDaddy，轉而跳到下一家 name.com 來進行申請。

不可否認 name.com 的配套相對來的少，其 coupon 優惠也很單純（privacy free），但是申請的過程很流暢，沒有過多不必要的資訊，網站本身速度也很快，會員後台的操作也很清楚明瞭，在 DNS 設定上還提供不少的 template 給人參考，我自己是比較喜歡這種單純直白的操作環境，即使價格不是最漂亮的。

最後我也順利申請到想要的新域名 hansliu.com 。

順便分享在搜尋域名資料時，看到的可愛影片，是介紹 .me 這個頂級網域名稱。
