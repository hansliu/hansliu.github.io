洗照片的故事

Hans Liu on 2014-04-27 category 樂分享 tags 底片 comments Comments fb Comments

整理自己去過的相機店

台北市

全彩現象

地址：台北市八德路二段229號2樓
電話：(02)2751-9235
正沖 + 上夾 + 底掃 要價190
Sample：
使用相機：Canon AE-1 + 50mm f1.8
底片：Kodak ELITE CHROME ISO 100

沖正片的品質相當的好，雖然老闆阿發的個性還挺生冷的，但是對於沖片的專業和技術卻是無話可說，底掃只有三百萬畫素，底掃出來的品質普普通通，尚可接受。

台北縣

永和華國

地址：台北縣永和市中興街5號
電話：(02)2925-0022 (02)2928-0922
負沖 + 底掃(畫素低) 要價130
Sample：
使用相機：Vivitar 22mm 光圈f11 快門1/125s
底片：Fuji X-TRA ISO 400

剛開始拍 Vivitar 的時候去過幾次，可以正沖負是我對這家店的印象，不過目前還沒有機會正沖負，以後想要嘗試看看，沖片出來給我的感覺是偏藍色調，品質可以接受，但是並無特別突出的地方，底掃只有三百萬畫素是蠻可惜的一點。

新莊麗來

地址：24245 台北縣新莊市中港路314號1樓
電話：(02)2996-1686
負沖 + 底掃 要價120
負沖 + 半格底掃 要價140
Sample：
使用相機：Tura 28mm 光圈f9 快門1/125s
底片：Fuji X-TRA ISO 400

Sample:
使用相機：Olympus pen EED
底片：Fuji X-TRA ISO 400

慕名以久的店家，給麗來沖片的次數還不多，不過給人的感覺相當的專業和有水準，沖掃的品質讓我很滿意，不過底掃顏色感覺相當艷麗，不知道是否是店家調整的口味較重，讓人印象深刻的地方在於，沖好的底片都剪裁的相當整齊並且用無酸袋裝好，讓人體會的到店家的用心。

淡水百合

地址：台北縣淡水鎮中山路87號
電話：
負沖 + 底掃 要價150
Sample：
使用相機：Vivitar 22mm 光圈f11 快門1/125s
底片：Fuji X-TRA iso400

我在淡水沖洗的店家，沖片品質穩定，底掃也接受高畫素，底掃品質尚可接受，假如在淡水不想跑太遠，百合不妨是一個好選擇。
