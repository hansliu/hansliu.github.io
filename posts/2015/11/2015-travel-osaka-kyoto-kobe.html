2015大阪京都神戶七天六夜

Hans Liu
on 2015-11-20 category 吃喝玩樂 tags 日本 大阪 京都 神戶 comments Comments fb Comments

這次大阪京都神戶之旅是我自由行以來規劃最大團的一次，而且行程的一開始就包含各種變數，有的人是要從香港出發到日本跟我們會合，有的人則是因為假期關係要提早回台灣，也因為這樣導致大家坐飛機的時間都不同，所以要感謝大家一路上的配合，讓我們得以順利的完成這趟旅行。

在景點規劃上，七天的行程主要包含一天的神戶、一天半的大阪和兩天半的京都，稍微美中不足的部分就是去摩耶山那天的天氣狀況很差，沒有夜景可看，其次就是因為關西這次冷的很慢，所以楓葉還沒變紅，沒有機會欣賞到滿山滿谷的紅葉。

七天的行程大綱與有印象的吃喝

  • Day 1
    關西機場 -> 南海電鐵 -> 飯店 -> 日本橋 -> 難波/道頓堀/心齋橋 -> 飯店
    吃飛機餐、大阪燒、章魚燒、迴轉壽司
  • Day 2
    JR West Pass -> 姬路城 -> 神戶三宮 -> 摩耶山下站 -> 神戶港 -> 難波/道頓堀/心齋橋 -> 飯店
    吃飯店早餐、松屋、肉包、章魚燒
  • Day 3
    大阪海遊館周遊券 -> 今昔生活館 -> 天滿橋商店街 -> 大阪海遊館 -> 難波/道頓堀/心齋橋 -> 國產和牛燒肉 -> 飯店
    吃飯店早餐、唐揚玉子燒定食、和牛燒肉
  • Day 4
    大阪京都一日券 -> 京阪本線 -> 伏見稲荷大社 -> 嵐山 -> 京都祇園河原町 -> 京都祇園四條通 -> 花見小路通 -> 鳥貴族 -> 飯店
    吃飯店早餐、醬油糰子、肉包、蕨餅抹茶冰、關東煮、湯豆腐、麥芽糖糰子、居酒屋
  • Day 5
    大阪市地鐵 -> 大阪萬博公園 -> 難波/道頓堀/心齋橋 -> 飯店 -> 京阪本線 -> 河原町 -> 飯店 -> 貴船 -> 飯店
    吃飯店早餐、散壽司丼飯
  • Day 6
    京阪本線 -> 鴨川神社 -> 錦市場 -> 宇治 -> 難波/道頓堀/心齋橋 -> 梅田 -> 飯店
    吃飯店早餐、章魚鵪鶉蛋、現打果汁、近江牛肉串、干貝串、煎餅、蟹肉、中村藤吉抹茶大福紅豆湯、宇治抹茶冰、一蘭、麥當勞
  • Day 7
    大阪市地鐵 -> 大阪城 -> 大阪NHK電視台 -> 阿倍野近鐵百貨 -> 難波/道頓堀/心齋橋 -> 飯店 -> 南海電鐵 -> 關西機場
    吃壽司
